# ccpkolkata

```Made for Assignment Purposes``` 

Student Made site of http://www.polytechnic.wbtetsd.gov.in/centralcalpoly

### :hammer_and_wrench: Building

- Prerequiste: Node >= 14.16.1, Yarn package manager
- Run `yarn` to install required dependencies.
- Run `yarn start` or `yarn build` to run or build.

### :star2: Tech Used

- HTML5
- CSS3
- Node
- Parcel Bundler
